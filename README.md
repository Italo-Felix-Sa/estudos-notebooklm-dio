# estudos-notebooklm-dio
Exploração do ecossistema Open Source: governança, licenciamento e o impacto econômico. Analisa as filosofias FSF/OSI, o modelo Catedral vs. Bazar  e o poder do desenvolvimento colaborativo. Inclui miniguia com glossário e prompts para otimizar a aprendizagem em engenharia de software e sustentabilidade FOSS.
